Keyboard Shortcuts

  • Tab: Restart test
  • Esc: Cancel/exit test
  • Ctrl + Enter: Submit (where applicable)
  • Ctrl + L: Focus typing input
  • Ctrl + Shift + D: Open dashboard
  • Ctrl + Shift + Q: Switch to quote mode
  • Ctrl + Shift + C: Switch to code mode
  • Ctrl + Shift + P: Switch to practice mode
  • Ctrl + Shift + F: Switch to finger training
  • Ctrl + Shift + L: Switch to lessons

Tip: You can use Tab to quickly restart a test at any time.
